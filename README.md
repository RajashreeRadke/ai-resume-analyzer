# AI Resume Analyzer

A full-stack application that analyzes resumes using AI.

## 🚀 Tech Stack
- React.js (CoreUI)
- Node.js + Express
- Gemini API

## ✨ Features
- Paste resume text
- AI-based analysis
- Feedback: strengths, weaknesses, suggestions
- ATS score

## 📦 Setup

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
